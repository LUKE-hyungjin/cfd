# 경계층

{{% hint info %}}
**선수지식**: [Navier-Stokes 방정식](/ko/docs/fundamentals/fluid-mechanics/navier-stokes) | [층류와 난류](/ko/docs/fundamentals/fluid-mechanics/laminar-turbulent)
{{% /hint %}}

## 한 줄 요약
> **벽 바로 옆은 느리고, 멀어질수록 빨라진다. 이 속도가 변하는 얇은 영역이 경계층이다.**

## 왜 알아야 하나?

경계층은 CFD에서 **가장 중요한 영역** 중 하나입니다.

이유:
1. **벽면전단응력(WSS)**: 경계층에서 결정됨 → 동맥경화 예측
2. **격자 설계**: 경계층에 격자를 잘 배치해야 정확한 결과
3. **혈류 역학**: 혈관 벽 근처 현상 이해 필수

## 왜 경계층이 생기나?

### 점착 조건 (No-slip Condition)

유체가 벽에 닿으면:

> **벽에 붙은 유체의 속도 = 벽의 속도 (보통 0)**

이건 실험적으로 확인된 사실입니다. 유체 분자가 벽면에 "붙어서" 움직이지 않습니다.

### 비유: 달리는 버스

버스 안의 공기를 생각해보세요:
- 버스 벽에 붙은 공기: 버스와 같이 움직임 (상대 속도 0)
- 버스 중앙의 공기: 관성으로 약간 뒤처짐

벽 근처와 중앙의 속도 차이 → 이게 경계층

### 경계층의 형성

```
자유류 속도 U∞ →→→→→→→→→→→→
                 ↗↗↗↗↗↗↗↗↗↗  ← 경계층 (속도 변화 영역)
벽면 속도 = 0   ═══════════════
```

- **벽면**: 속도 = 0 (점착 조건)
- **경계층 내부**: 0에서 서서히 증가
- **경계층 밖 (자유류)**: 일정한 속도 $U_\infty$

## 경계층 두께

### 정의

경계층 두께 $\delta$: 속도가 자유류 속도의 99%에 도달하는 거리

$$
u(y = \delta) = 0.99 \times U_\infty
$$

### 층류 경계층 (Blasius 해)

평판 위 층류 경계층:

$$
\delta = \frac{5x}{\sqrt{Re_x}}
$$

- $x$ : 평판 시작점부터의 거리
- $Re_x = \frac{U_\infty x}{\nu}$ : 국소 레이놀즈 수

**특징**:
- 앞에서 뒤로 갈수록 두꺼워짐
- Re가 크면 얇아짐

### 혈관에서의 경계층

혈관은 평판이 아니라 **원형 관**이므로 좀 다릅니다.

완전 발달 층류 (Poiseuille 흐름)에서:

$$
u(r) = U_{max} \left( 1 - \frac{r^2}{R^2} \right)
$$

- $r$ : 중심으로부터 거리
- $R$ : 관 반지름
- $U_{max}$ : 중심 최대 속도

```
      U_max
        ↓
    ────────────  ← 속도 프로필 (포물선)
   /            \
  /              \
 │      ●        │  ← 관 중심
  \              /
   \            /
    ────────────
        0
```

**특징**:
- 중심에서 최대, 벽에서 0
- 포물선 모양 (층류일 때)

## 벽면전단응력 (Wall Shear Stress)

### 정의

경계층 때문에 벽면에 마찰력이 발생합니다:

$$
\tau_w = \mu \left. \frac{\partial u}{\partial y} \right|_{y=0}
$$

- $\tau_w$ : 벽면전단응력 (Pa)
- $\mu$ : 점도
- $\frac{\partial u}{\partial y}|_{y=0}$ : 벽면에서의 속도 기울기

### 물리적 의미

```
     빠름 →→→→→→
            ↑ 속도 차이 = 기울기
     느림 →→
═══════════════  벽면
```

속도 기울기가 클수록 → 벽면전단응력이 큼

### Poiseuille 흐름에서 WSS

$$
\tau_w = \frac{4 \mu Q}{\pi R^3}
$$

- $Q$ : 유량

**의미**:
- 점도↑ → WSS↑
- 유량↑ → WSS↑
- 반지름↓ (혈관 좁아짐) → WSS↑↑↑ (세제곱 영향!)

## 생체유체에서 WSS의 중요성

WSS는 심혈관 질환의 핵심 지표입니다:

| WSS 범위 | 상태 | 결과 |
|----------|------|------|
| 1-7 Pa | 정상 | 건강한 내피세포 |
| < 0.4 Pa | 낮음 | 동맥경화 위험↑ |
| > 7 Pa | 높음 | 혈전 위험↑ |

**왜?**
- **낮은 WSS**: 흐름이 정체 → 콜레스테롤 쌓임 → 동맥경화
- **높은 WSS**: 내피세포 손상 → 혈전 형성

### 동맥류에서

동맥류(혈관이 부풀어오른 곳)에서:

```
정상 혈관        동맥류
  ====>        /‾‾‾‾‾\
 높은 WSS      낮은 WSS (정체)
                \_____/
```

- 동맥류 안: 흐름 정체, 낮은 WSS
- 낮은 WSS → 벽이 약해짐 → 파열 위험↑

## CFD에서 경계층 처리

### 격자 설계

경계층을 정확히 계산하려면 **벽 근처에 격자를 촘촘히** 배치해야 합니다:

```
═══════════════════════  벽
─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   ← 얇은 층
──────────────────────
──────────────────────
──────────────────────   ← 멀어질수록 넓게
          ↑
      경계층 격자
```

### y+ (와이 플러스)

벽 근처 격자 품질을 나타내는 지표:

$$
y^+ = \frac{y u_\tau}{\nu}
$$

- $y$ : 벽에서 첫 번째 격자 중심까지 거리
- $u_\tau = \sqrt{\tau_w / \rho}$ : 마찰 속도

| 해석 방법 | 권장 y+ |
|----------|---------|
| 벽 함수 (Wall Function) | 30 ~ 300 |
| 저레이놀즈 모델 | < 1 |

생체유체 CFD에서는 WSS가 중요하므로 보통 **y+ < 1**을 목표로 합니다.

## 정리

| 개념 | 설명 |
|------|------|
| 경계층 | 벽 근처 속도가 변하는 얇은 영역 |
| 점착 조건 | 벽에서 유체 속도 = 0 |
| WSS | 벽면에 가해지는 마찰력, 심혈관 핵심 지표 |
| y+ | 벽 근처 격자 품질 지표 |
