# 유체역학

유체의 운동을 지배하는 기본 원리와 방정식입니다.

## 학습 순서

1. [유체란 무엇인가](/ko/docs/fundamentals/fluid-mechanics/what-is-fluid) - 유체의 정의와 특성
2. [연속 방정식](/ko/docs/fundamentals/fluid-mechanics/continuity) - 질량 보존 법칙
3. [Navier-Stokes 방정식](/ko/docs/fundamentals/fluid-mechanics/navier-stokes) - 유체 운동의 핵심
4. [층류와 난류](/ko/docs/fundamentals/fluid-mechanics/laminar-turbulent) - 두 가지 흐름 패턴
5. [경계층](/ko/docs/fundamentals/fluid-mechanics/boundary-layer) - 벽면 근처의 유동

## 왜 배워야 하나?

CFD(전산유체역학)는 컴퓨터로 유체의 흐름을 시뮬레이션하는 기술입니다. 시뮬레이션을 하려면 먼저 **유체가 어떤 법칙을 따르는지** 알아야 합니다.

자동차가 어떻게 움직이는지 모르면 자동차 게임을 만들 수 없듯이, 유체가 어떻게 움직이는지 모르면 CFD를 할 수 없습니다.
